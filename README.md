# JAX-Haiku-GAN
This project contains the JAX-Haiku implementations of some GAN variations, and is mainly developed using Kaggle. The dataset used in this project is CelebA.

## Dependencies
* JAX
* Haiku
* Optax

## List of contents

2. WGAN (to do)
